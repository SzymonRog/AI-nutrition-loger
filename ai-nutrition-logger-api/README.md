# AI Nutrition Logger

An AI-powered application for logging and analyzing nutrition data.

## Project Structure

```
ai-nutrition-logger/
├── assets/           # Static assets (icons, images)
├── config/           # Configuration files
├── data/             # Data storage
│   ├── raw/         # Raw input data
│   └── processed/   # Processed data
├── docs/            # Documentation
├── logs/            # Application logs
├── notebooks/       # Jupyter notebooks for analysis
├── src/             # Source code
│   ├── api/         # API endpoints
│   ├── core/        # Core business logic
│   ├── models/      # Data models
│   └── processors/  # Data processors
└── tests/           # Test files
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python -m src.main
```
